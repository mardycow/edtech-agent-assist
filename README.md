# **EdTech Agent Assist**

## **Описание**

AI-система поддержки образовательных платформ, которая в реальном времени генерирует черновики ответов для операторов и админов, работая с различными категориями запросов как в чатах поддержки, так и опционально в других точках взаимодействия со студентом. Предназначена для снижения нагрузки на специалистов и предоставления верифицированных ответов на основе единой корпоративной базы знаний.
