## **6-7. Расчет необходимой памяти для выбранных моделей (обоснование выбора)**

Используемые формулы:

- Для загрузки параметров модели: _[число параметров] x [память на 1 параметр] x 1,2_
- Для KV-кэша (с учетом архитектуры Qwen3.5, для 1 батча):

_2 x [число слоев gated attention] x [число голов kv] x [размерность головы (head_dim)] x [число байт на токен] x [длина контекста]_

### **RTX 4090**

**Qwen3.5-4B**:

- веса модели: _4 x 2 x 1,2 = **9.6 Гб**_
- KV-кэш: _2 x 8 x 4 x 256 x 2 x 128 000 **~=3.9 Гб**_

(с тем же значением кэша модель 9B (21.6 Гб) уже не развернуть на 24 Гб)

**Qwen3.5-35B-A3B**:

- веса модели: _35 x 0.5 x 1,2 = **21 Гб**_
- KV-кэш: _2 x 10 x 2 x 256 x 0.5 x 128 000 **~=0.6 Гб**_

### **H100**

**Qwen3.5-27B**:

- веса модели: _27 x 2 x 1,2 = **64.8 Гб**_
- KV-кэш: _2 x 16 x 4 x 256 x 2 x 128 000 **~=7.8 Гб**_

(для модели 35B даже на веса места не хватит)

**Qwen3.5-122B-A10B**:

- веса модели: _122 x 0.5 x 1,2 = **73.2 Гб**_
- KV-кэш: _2 x 12 x 2 x 256 x 0.5 x 128 000 **~=0.7 Гб**_
