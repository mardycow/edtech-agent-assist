# **Выбор модели для AI-системы**

## **1. Жесткие атрибуты и их ограничения**

1. **Происхождение данныхи авторские права**: модель должна иметь открытые веса и лицензию, допускающую свободное использование.

2. **Производительность**: генерация текста должна начинаться не позднее 1-3 секунды после запроса (т.к. система направлена на помощь в реальном времени).

3. **Функциональность**:

- **поддержка русского языка**: важно высокое качество понимания специфики русского языка (использование в рамках русскоязычной платформы);
- **длина контекста**: не менее 32k токенов (для реализации RAG и поддержки контекста диалога);

4. **Контроль**: модель должна генерировать ответы строго на основе предоставленной базы знаний и соблюдать заданный формат (высокая способность следовать инструкциям).
