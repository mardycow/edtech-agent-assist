# **Критерии оценки AI-систем**

## **1. Бизнес-цель**

_"Какую проблему решает AI-система?"_

_**Снижение нагрузки** на операторов поддержки и сотрудников, работающих в иных каналах взаимодействия со студентами (администарторы, кураторы) и **обеспечение согласованности предоставляемых ответов** в рамках всей платформы (за счет генерации черновиков ответов на основе единой базы знаний и автоматизации ответов на вопросы по заранее известным сценариям)_ --->

---> **Бизнес-цель (по SMART)**: Сократить среднее время обработки обращений на 30% и снизить количество повторных обращений, вызванных некорректными ответами (жалобы), в 2 раза в течение 6 месяцев.

## **2. Декомпозиция бизнес-цели в наблюдаемые результаты**

Достижение цели выражается в:

- снижении среднего времени обработки запроса;
- увеличении количества принятых черновиков ответов;
- повышении согласованности ответов с источниками;
- увеличении точности классификации категорий запросов.

## **3. Определение критериев оценки**

1. Снижение среднего времени обработки -> **Задержка**
2. Увеличинение количества принятых черновиков -> **Способность модели следовать инструкциям**
3. Повышение согласованности с источниками -> **Способность к генерации (фактическая согласованность)**
4. Увеличение точности классификации категорий запросов -> **Предметно-ориентированная способность**

## **4. Шкала оценки критериев**

**Задержка**:

- **1** - <3 сек.
- **2** - 5-8 сек.
- **3** - >10 сек.

**Способность следовать инструкциям**:

- **1** - отправлено без правок
- **2** - незначительные стилистичсекие правки
- **3** - частичное исправление в семантике (добавление фактов)
- **4** - полное исправление ответа (ошибки, галлюцинации)

**Фактичеcкая согласованность**:

- **1** - без галлюцинаций и противоречий (текст подтвержден ссылкой)
- **0** - противоположное (ссылка не подтверждает текст)

**Предметно-ориентированная способность**:

- **1** - верная классификация
- **0** - неверная классификация
